# Quick Guide

# Command Syntax

`& = Arg is not required.`

`*?new [type] [name] [&Display] [&SortOrder]*`
`*?remove [type]*`

*Defaults to sidebar and ascending order when not given.*

# Simple

`*?new deaths Deaths%Scoreboard*`
`*?remove deaths*`

`*?new brokenBlocks Blocks%Broken BelowName*`
`*?remove brokenBlocks*`
